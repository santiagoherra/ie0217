# README.md

**Este es el repositorio del curso ie0217, estructuras abstractas de datos y algoritmos para ingenieria.**

Para de este curso es entender el funcionamiento de la libreria *Git* y la pagina web que da sus servicios *Github*.

**Algunos de sus comandos importantes son:**

```
git status
```
Para poder ver los cambios realizados en un archivo.

```
git add
```
Para poder pasar los cambios la zona de revision (stage).

```
git commit
```
Para que los cambios se ejecuten en el repositorio local.

```
git push
```
Para que los cambios se ejecuten en el repositorio remoto.
